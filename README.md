# Dev environment

1. Clone the proyecto
2. Copy ```env.template``` and rename it to ```.env```
3. Run
  ```yarn install```
4. Set up docker image
  ```docker-compose up -d```
5. Run
  ```tarn start:dev```
6. Navigate to ```http://localhost:3000/graphql```
